# دليل التراجع السريع

## في حال حدث خطأ في التعديلات الأخيرة

### الرجوع للنسخة المستقرة الأخيرة (v1.0):

```bash
git reset --hard v1.0-stable
git push --force origin main
```

### رؤية كل النسخ المحفوظة:

```bash
git tag -l
```

### رؤية تفاصيل نسخة معينة:

```bash
git show v1.0-stable
```

## حفظ نسخة جديدة (بعد كل تطوير ناجح)

```bash
git tag -a v1.X-stable -m "وصف التحديث"
git push origin v1.X-stable
```

## الاستراتيجية

١. اشتغل عادي على `main`.
٢. لما تخلص ميزة كبيرة وتشتغل تمام، احفظ tag جديد.
٣. لو شي صار غلط، استخدم أوامر الرجوع أعلاه.
٤. GitHub يحفظ كل شي، ما تخسر شي.
