# تطبيق المعلم الذكي — النموذج الأولي

تطبيق سعودي للمعلمين يجمع **سجل الطلاب** + **توليد المحتوى بالذكاء الاصطناعي** + **ملف الإنجاز التلقائي**.
نموذج HTML أولي يعمل في المتصفح بدون خادم.

---

## 🚀 التشغيل

```bash
cd teacher_app
python3 -m http.server 8000
# افتح: http://localhost:8000
```

لأي متصفح حديث (Chrome / Safari / Edge / Firefox).

## 🤖 تفعيل الذكاء الاصطناعي

1. احصل على مفتاح من [console.anthropic.com/settings/keys](https://console.anthropic.com/settings/keys).
2. في التطبيق: ⚙️ الإعدادات → ألصق المفتاح → اختبر الاتصال.
3. بدون مفتاح: التطبيق يعمل بنماذج تجريبية لاختبار الواجهة.

⚠️ **ضع حداً شهرياً** في لوحة Anthropic لتجنب تكاليف غير متوقعة.

## 💾 البيانات

- كل شيء محلي في **IndexedDB** (متصفحك فقط).
- لا شيء يُرسل لأي خادم (عدا طلبات AI حين تُفعّلها).
- **نسخ احتياطي:** الإعدادات → "📤 تصدير جميع البيانات" (ملف JSON).
- **استيراد:** الإعدادات → "📥 استيراد نسخة".

## 🎨 الهوية البصرية

- **الألوان:** أزرق ملكي `#1E40AF` + أزرق فاتح `#3B82F6` + سماوي `#0EA5E9`
- **الخط:** Tajawal من Google Fonts
- **الاتجاه:** RTL بالكامل
- **المسافات:** مقياس ثابت (٤، ٨، ١٢، ١٦، ٢٠، ٢٤، ٣٢، ٤٠...)

## 🔧 التقنيات

- HTML5 + CSS3 + JavaScript vanilla (بدون frameworks)
- IndexedDB للتخزين المحلي
- Anthropic Claude API (اختياري)
- `window.print()` مع `print.css` لتوليد PDF

---

## 📋 قيود النسخة الأولية

- لا توجد مزامنة سحابية (بيانات محلية لكل متصفح).
- لا يوجد بوابة طالب/ولي أمر.
- OCR للصور غير مدعوم (إدخال يدوي للأسماء والنصوص).
- لا توجد إشعارات push فعلية (التنبيهات محسوبة داخل التطبيق فقط).
